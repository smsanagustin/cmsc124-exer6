# cmsc124-exer6
private repo for the exercise 6 in cmsc 124 (design of programming languages)

# how to run 
### on 2 separate terminals, run this:
`erlang -name <name>@<host>`
`erlang -name <name>@<host>`

### then run on separate terminals:
### terminal 1:
`c(chat).`
`chat:init_chat().`

### terminal 2:
`c(chat).`
`chat:init_chat2('<name1>@<host>')`
